Installation Instructions for FoundryVTT

  1. Copy the entire strongholds-and-followers folder to your FoundryVTT Data/modules/ directory
  2. Restart FoundryVTT
  3. Enable the module in your world's module settings
  4. GM: Look for the castle icon in scene controls
  5. Players: Look for "Strongholds" button in player list

Usage

For GMs:

  • Click the castle icon in scene controls to open Stronghold Manager
  • Create new strongholds with descriptions
  • Upgrade strongholds (costs gold)
  • Add custom bonuses from the PDF

For Players:

  • Click "Strongholds" button in player list to view active strongholds
  • See stronghold levels, costs, and bonuses
  • Bonuses will appear in chat during extended rests (if enabled)
